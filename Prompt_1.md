# Gemini 繪圖 Prompt - 工作流程助理架構圖

> **用途**：使用 Gemini 生成「工作流程助理」的資料流向架構圖  
> **日期**：2026-03-04

---

## 🎨 Gemini Prompt

```
請繪製一個專業的系統架構流程圖，展示「工作流程助理」AI Agent 的兩個核心流程。

【圖表要求】
- 風格：現代化、科技感、清晰易懂
- 配色：使用藍色系（主要）、綠色（成功）、橙色（強調）
- 排版：垂直或水平佈局，流程清晰
- 圖示：使用現代化的 icon（對話、資料庫、流程等）

【流程 1：資料輸入流程】（左側或上方）

標題：「📝 資料填寫流程」

流程步驟：
1. 「使用者」（人物 icon）
   ↓ 對話互動
2. 「Copilot Studio Agent」（AI 對話 icon）
   - 名稱：工作流程助理
   - 功能：引導式對話收集
   ↓ 呼叫 Power Automate Flow
3. 「Power Automate Flow」（齒輪/自動化 icon）
   - Flow 1：建立工作主檔
   - Flow 2：建立流程步驟
   ↓ 寫入資料
4. 「SharePoint Lists」（資料庫/列表 icon）
   - List A：工作項目主檔
   - List B：工作流程步驟
   
在此流程中標註關鍵點：
- 步驟 1→2：「自然語言對話」
- 步驟 2→3：「結構化資料傳遞」
- 步驟 3→4：「自動儲存」

【流程 2：知識庫問答流程】（右側或下方）

標題：「💬 知識庫問答流程」

流程步驟：
1. 「使用者」（人物 icon）
   - 提出問題：「XXX 流程怎麼做？」
   ↓ 問題傳遞
2. 「Copilot Studio Agent」（AI 對話 icon）
   - 名稱：工作流程助理
   ↓ 觸發查詢 Topic → 呼叫 Flow
3. 「Power Automate Flow」（搜尋/查詢 icon）
   - Flow 3：Search_WorkFlow
   ↓ 查詢 SharePoint
4. 「SharePoint Lists」（資料庫/列表 icon）
   - 搜尋關鍵字
   - 撈取相關流程 + 步驟
   ↓ 回傳資料
5. 「Power Automate Flow」（處理 icon）
   - 組合成格式化文字
   - 包含：流程名稱、部門、步驟說明
   ↓ 回傳給 Agent
6. 「Copilot Studio Agent」（AI 對話 icon）
   - 以自然語言呈現
   ↓ 友善回覆
7. 「使用者」（人物 icon）
   - 收到清楚的流程說明

在此流程中標註關鍵點：
- 步驟 1→2：「自然語言提問」
- 步驟 2→3：「參數化查詢」
- 步驟 4→5：「結構化資料」
- 步驟 5→6：「格式化回應」
- 步驟 6→7：「自然語言回覆」

【特殊要求】
1. 兩個流程之間用虛線框分隔
2. SharePoint Lists 在兩個流程中是同一個資料庫（用箭頭或連線表示共用）
3. 強調「資料填寫」是寫入（Write），「知識問答」是讀取（Read）
4. 在圖表底部加入說明文字：
   「資料流向：使用者對話 → AI Agent 處理 → Power Automate 自動化 → SharePoint 儲存/查詢」

【技術標籤】（顯示於圖表角落）
- Microsoft Copilot Studio
- Power Automate
- SharePoint Lists
- 自然語言處理 (NLP)

請繪製清晰、專業且易於理解的架構圖。
```

## 📋 補充說明

### 如果 Gemini 需要更多細節，可補充：

**流程 1 細節補充：**
```
使用者對話範例：
- Agent 問：「工作項目名稱是什麼？」
- 使用者答：「產品出貨流程」
- Agent 問：「第一個步驟是什麼？」
- 使用者答：「登入 SAP 系統」

Agent 對話節點：
- Topic 1：收集主檔資訊
- Topic 2：收集步驟資訊（Loop）

Power Automate 功能：
- 將對話資料轉換為 SharePoint 欄位
- 建立主檔後取得 ID，供步驟關聯
```

**流程 2 細節補充：**
```
查詢範例：
- 使用者問：「產品出貨流程怎麼做？」
- Flow 搜尋：Title 包含「產品出貨」
- 撈取結果：主檔資訊 + 所有步驟
- 組合格式：
  📋 產品出貨流程
  步驟 1：登入 SAP 系統
  步驟 2：建立出貨單
  ...
- Agent 回覆：以友善語氣呈現
```

│  使用者             │    │  使用者             │
│    ↓               │    │    ↓               │
│  Agent             │    │  Agent             │
│    ↓               │    │    ↓               │
