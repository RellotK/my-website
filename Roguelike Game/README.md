# 🗡️ 深淵探險者 - Roguelike RPG

一款充滿挑戰與策略的 Roguelike 類型網頁 RPG 遊戲。探索無盡的地牢，擊敗強大的敵人，收集珍稀裝備，成為最強的深淵探險者！

## 🎮 遊戲特色

### 核心玩法
- **隨機生成地牢**：每次遊戲都有不同的地圖佈局
- **回合制戰鬥**：策略性的戰鬥系統，需要謹慎選擇行動
- **角色成長**：擊敗敵人獲得經驗值，提升等級強化屬性
- **裝備系統**：收集武器、護甲和飾品來增強角色能力
- **無盡挑戰**：地牢深度越深，敵人越強，獎勵也越豐富

### 遊戲系統

#### 📊 角色屬性
- **生命值（HP）**：角色的生存能力
- **攻擊力**：決定造成的傷害
- **防禦力**：減少受到的傷害
- **暴擊率**：造成雙倍傷害的機率
- **經驗值**：累積經驗升級

#### ⚔️ 戰鬥系統
1. **普通攻擊**：使用當前攻擊力對敵人造成傷害
2. **防禦姿態**：大幅減少受到的傷害
3. **特殊技能**：消耗生命值造成 1.8 倍傷害
4. **逃跑**：40% 機率逃離戰鬥

#### 🎒 裝備與物品
- **武器**：增加攻擊力（鐵劍、鋼劍、傳說之劍等）
- **護甲**：增加防禦力（皮甲、鎖甲、龍鱗甲等）
- **飾品**：提供綜合屬性加成（力量戒指、守護護符等）
- **藥水**：恢復生命值
- **金幣**：用於治療或未來功能擴展

#### 👹 敵人種類
遊戲中有 9 種不同的敵人，隨著地牢深度增加會遇到更強大的對手：
- 哥布林（等級 1）
- 骷髏戰士（等級 2）
- 黑暗法師（等級 3）
- 獸人勇士（等級 4）
- 石像鬼（等級 5）
- 惡魔（等級 6）
- 吸血鬼（等級 7）
- 龍族守衛（等級 8）
- 深淵領主（等級 9）

## 🎯 操作說明

### 移動控制
- **滑鼠點擊**：點擊相鄰格子移動
- **WASD 鍵**：W（上）、S（下）、A（左）、D（右）
- **方向鍵**：↑↓←→ 控制移動

### 按鈕功能
- **🎮 新遊戲**：開始新的冒險
- **⬇️ 下一層**：找到樓梯後進入下一層地牢
- **💊 治療**：花費 30 金幣恢復 50% 生命值
- **📖 技能樹**：打開技能樹面板，學習和裝備技能

### 戰鬥操作
進入戰鬥後可選擇：
- **⚔️ 攻擊**：對敵人造成傷害
- **🛡️ 防禦**：減少受到的傷害
- **🏃 逃跑**：嘗試逃離戰鬥
- **技能按鈕**：使用已裝備的技能（消耗魔力）

### 技能樹操作
- **學習技能**：使用技能點學習新技能或升級現有技能
- **自動使用**：學習後的技能會自動在戰鬥中顯示為按鈕

## 🎨 遊戲元素

### 地圖符號
- 🧙 **玩家**：你控制的角色
- 👹 **敵人**：需要戰鬥的怪物
- 💎 **物品**：可以撿取的寶物
- 🚪 **樓梯**：通往下一層的入口
- 🧱 **牆壁**：無法通過的障礙

### 物品稀有度
- **普通（灰色）**：基礎物品
- **優秀（綠色）**：較好的裝備
- **稀有（藍色）**：強力裝備
- **傳說（金色）**：最強裝備

## 💡 遊戲技巧

1. **謹慎探索**：不要急於深入，先在當前層提升等級
2. **裝備優先**：遇到裝備優先撿取，能大幅提升戰力
3. **保持血量**：及時使用治療或撿取藥水，避免血量過低
4. **戰鬥策略**：血量充足時主動攻擊，血量低時使用防禦或嘗試逃跑
5. **資源管理**：合理使用金幣和魔力，不要浪費在不必要的治療上
6. **小心陷阱**：移動時有 5% 機率觸發陷阱，損失 10% 最大生命值
7. **技能規劃**：優先學習和升級常用技能，合理分配技能點
8. **技能升級**：提升技能等級可以增強效果，比學習更多技能更有效率
9. **魔力管理**：注意技能消耗，保留足夠魔力應對危機
10. **裝備掉落**：高等級敵人掉落更好裝備，戰勝強敵收益更高

## 🚀 如何開始

1. 在瀏覽器中打開 `index.html` 文件
2. 點擊「新遊戲」按鈕開始冒險
3. 使用滑鼠或鍵盤控制角色移動
4. 探索地牢，擊敗敵人，收集裝備
5. 找到樓梯進入更深的地牢層級
6. 挑戰自己能達到的最大深度！

## 🛠️ 技術特點

- **純前端實現**：使用原生 HTML、CSS、JavaScript 開發
- **無需伺服器**：直接在瀏覽器中運行
- **響應式設計**：支援不同螢幕尺寸
- **流暢動畫**：CSS 動畫提供視覺反饋
- **模組化代碼**：清晰的代碼結構，易於維護和擴展

### 🌟 技能樹系統
- **技能點獲得**：每次升級獲得 1 點技能點
- **技能學習**：使用技能點學習和升級技能
- **自動顯示**：學習後的技能會自動在戰鬥中顯示
- **職業技能**：每個初始職業都有 4 個獨特技能樹

#### 戰士技能
- **重擊**：基礎攻擊技能，可升至 Lv.5
- **鐵壁防禦**：大幅提升防禦力 3 回合
- **戰吼**：降低敵人攻擊力
- **順劈斬**：無視部分防禦的強力攻擊

#### 法師技能
- **魔法飛彈**：基礎魔法傷害
- **寒冰箭**：造成傷害並減緩敵人
- **魔力護盾**：吸收傷害的護盾
- **閃電鏈**：高傷害魔法攻擊

#### 牧師技能
- **治療術**：恢復生命值
- **神聖之光**：對不死系敵人額外傷害
- **祝福**：提升攻擊力
- **神聖新星**：造成傷害並恢復生命

#### 盜賊技能
- **暗襲**：快速攻擊
- **毒刃**：持續中毒傷害
- **閃避步**：提升閃避機率
- **背刺**：超高暴擊傷害

### 💎 裝備掉落系統
- **掉落機率**：基礎 30%，敵人等級越高機率越大
- **品質分級**：
  - ⚪ 普通（50%）
  - 🟢 優秀（33%）
  - 🔵 稀有（15%）
  - 🟡 傳說（2-10%，取決於敵人等級）
- **戰利品**：擊敗敵人有機會獲得裝備
